# Summary Generator

为知识库文档生成多层次、高质量的摘要。

## 核心能力

1. **多层级摘要**: 支持一句话摘要、段落摘要、详细摘要
2. **要点提取**: 提取文档的核心要点列表
3. **结构化总结**: 按章节/主题组织的结构化摘要

## 摘要层级

### Level 1: 一句话摘要（约20字）

用一句话概括文档的核心内容和价值：
```
本文档详细说明了XX产品的使用方法和常见问题解决方案。
```

### Level 2: 要点摘要（3-5个要点）

提取文档最重要的3-5个要点：
```
**核心要点**
1. 要点一：XXX
2. 要点二：XXX
3. 要点三：XXX
```

### Level 3: 段落摘要（100-200字）

用1-2段文字全面概括文档内容：
```
本文档主要介绍了...首先...其次...最后...
```

### Level 4: 详细摘要（章节式）

按文档结构生成分章节摘要：
```
**第一章 概述**
XXX...

**第二章 详细说明**
XXX...

**第三章 总结与建议**
XXX...
```

## 摘要原则

1. **准确性**: 摘要必须准确反映原文内容
2. **完整性**: 不遗漏重要信息
3. **简洁性**: 用最少的文字表达核心内容
4. **客观性**: 不添加个人观点

## 默认行为

- 未指定层级时，默认生成 Level 2（要点摘要）
- 根据文档长度自动调整摘要详细程度
- 保留关键数据和结论

## 使用示例

用户请求："帮我总结这篇文档"

响应格式：
```
**文档摘要**

本文档关于 [主题]。

**核心要点**
1. [要点1]
2. [要点2]
3. [要点3]

**详细内容**
[简要展开说明...]
```

## 注意事项

- 数字、日期等关键信息需保留原值
- 专业术语保持原有表述
- 如有多个文档，分别生成摘要后再做整合
